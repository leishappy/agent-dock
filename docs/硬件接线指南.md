# AgentDock 硬件接线指南

## 物料清单

| 物料 | 型号 | 数量 | 备注 |
|------|------|------|------|
| 主控 | ESP32-S3 DevKitC-1 | 1 | N8R8 或 N8 (8MB Flash) |
| 屏幕 | ST7789 240×240 IPS | 1 | 1.3~1.54 寸 SPI 接口 |
| USB Hub | GL850G 模块 | 1 | USB 2.0 一拖四 |
| USB-C 母座 | 16P 贴片 | 1 | 连接电脑端 |

## ST7789 接线（SPI，7 线）

| ESP32-S3 | 杜邦线颜色建议 | ST7789 模块 | 说明 |
|----------|:----------:|-------------|------|
| **3.3V** | 红 | VCC / 3V3 | 供电（不要接 5V） |
| **GND** | 黑 | GND | 共地 |
| **GPIO 11** | 橙 | MOSI / SDA / DIN | SPI 数据 |
| **GPIO 12** | 黄 | SCLK / SCL / CLK | SPI 时钟 |
| **GPIO 10** | 绿 | CS / SS | 片选 |
| **GPIO 9** | 蓝 | DC / RS / A0 | 数据/命令 |
| **GPIO 14** | 紫 | RST / RESET | 复位 |
| **GPIO 2** | 白 | BLK / LED | 背光 PWM |

> 部分 ST7789 模块没有 RST 引脚 — 此时接 `-1`（不接），代码中 `TFT_RST=-1` 已处理。
>
> 如果模块没有独立引出 BLK 背光引脚，通常背光默认常亮，可以不接 GPIO 2。

## 引脚对照速查

```
        ESP32-S3 引脚一览（本工程使用）

   USB-OTG (GPIO19/20) ──── 接 GL850G 上行口
                ┌────────────┐
   3.3V    ────│           ║ │──── USB-C (电脑)
   GND     ────│           ║ │
   GPIO 19 ────│  ESP32-S3  │
   GPIO 20 ────│            │
   GPIO 2  ────│            │──── 背光 BLK
   GPIO 9  ────│            │──── TFT DC
   GPIO 10 ────│            │──── TFT CS
   GPIO 11 ────│            │──── TFT MOSI
   GPIO 12 ────│            │──── TFT SCLK
   GPIO 14 ────│            │──── TFT RST
                └────────────┘
```

> ESP32-S3 的 GPIO 19/20 是内置 USB-OTG 的 D+/D-。
> 这组 USB 用于：① 烧录固件、② USB CDC 串口通信（与 PC 端 Python 脚本交互）。

## GL850G USB Hub 接线（简化版）

```
  PC USB-C ────── GL850G ───── USB-A × 3 (外设)
                    │
                    └──── ESP32-S3 USB-OTG (用作 CDC 通信 + 烧录)
```

如果只需要验证屏幕功能，暂时不接 GL850G 也可以。
ESP32-S3 直接通过板载 USB-C 连电脑即可。

## 常见问题

| 现象 | 可能原因 |
|------|---------|
| 白屏无显示 | 背光已亮但 SPI 无通信 — 检查 MOSI/SCLK/CS/DC |
| 黑屏完全不亮 | 背光没通电 — 检查 BLK 或 3.3V/GND |
| 花屏/错位 | DC 或 RST 引脚接触不良 |
| 颜色异常 | RGB 顺序配置问题（ST7789 默认 BGR） |
| 串口无输出 | USB CDC 未正确配置，检查 `ARDUINO_USB_CDC_ON_BOOT=1` |
