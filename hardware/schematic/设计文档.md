## 三、自锁电源电路 (PWR_HOLD)

```
HLK-PM01 +5V
    │
    ├── [SS8550 PNP] E
    │         B ──[R1 10k]── GPIO21 (PWR_HOLD)
    │         C ── +5V_SWITCHED (供 AMS1117)
    │
    ├── [按钮] (开机触发) ── GND  (短按拉低 → PM01 启动)
    │
    ESP32 启动后 → GPIO21 LOW → PNP 导通 → 5V 锁定
    关机 → GPIO21 HIGH → PNP 截止 → 系统断电
```

## 六、HLW8032 电能计量电路

```
AC L ──────┬────────────── 到继电器
           │
      [R9 1MΩ] ─┬─ V2P (HLW8032 引脚 2)
      [R10 1MΩ]─┤
      [R11 1MΩ]─┤
      [R12 1MΩ]─┘
           │
      [R13 1kΩ]── GND (数字地)

AC N ──────┬────────────── 到插座
           │
      [采样电阻 2mΩ] ─┬─ V1P (HLW8032 引脚 3)
           │          │
           │     [R14 1kΩ]── GND
           │
           └── V1N (HLW8032 引脚 4)

HLW8032 引脚：
  1: VCC (+5V) ── C9 100nF + C10 10μF ── GND
  2: V2P (电压采样+)
  3: V1P (电流采样+)
  4: V1N (电流采样-)
  5: GND
  6: TX (UART TX) ── 经过电平转换 ── ESP32 GPIO8 (RX)
  7: PF (脉冲输出, 悬空)
  8: NC
```

**关键注意事项**：
1. HLW8032 是 5V 供电，TX 输出是 5V 电平
2. ESP32-S3 GPIO 是 3.3V 耐受
3. **必须加电平转换！** 方案：2.2kΩ + 3.3kΩ 电阻分压，或 74LVC1T45

```
HLW8032 TX (5V) ──[R15 2.2k]──┬── ESP32 GPIO8 RX
                               │
                          [R16 3.3k]
                               │
                              GND
(分压比 3.3/(2.2+3.3) = 0.6, 5V×0.6 = 3.0V ✅)
```

| 位号 | 元件 | 规格 | 封装 | 数量 |
|------|------|------|------|------|
| U5 | 电能计量 IC | HLW8032 | SOP-8 | 1 |
| R9-R12 | 分压电阻 | 1MΩ 1% 0805 | 0805 | 4 |
| R13 | 分压电阻 | 1kΩ 1% | 0603 | 1 |
| R14 | 电流采样电阻 | 1kΩ 1% | 0603 | 1 |
| RS1 | 采样电阻 | 2mΩ 1W (锰铜丝或2512) | 2512 | 1 |
| R15 | 电平转换 | 2.2kΩ | 0603 | 1 |
| R16 | 电平转换 | 3.3kΩ | 0603 | 1 |
| C9 | MLCC | 100nF 50V | 0603 | 1 |
| C10 | 钽电容 | 10μF 16V | 1206 | 1 |

**采样电阻替代方案**：找不到 2mΩ 的话，用 5mΩ 2512 (1W)，软件里改校准系数。

## 七、模块插槽连接器 (Pogo Pin)

每个插槽使用 **2×5 排母** (2.54mm)，模块上焊排针。打样阶段足够用了。

```
插槽 1-4 (PCB 上)：
┌──────────────┐
│ 1  VCC_5V   │  ← 5V 电源给模块
│ 2  GND      │
│ 3  SDA      │  ← I2C 数据（所有插槽共享）
│ 4  SCL      │  ← I2C 时钟（所有插槽共享）
│ 5  INT#1    │  ← 独立中断 (GPIO3/4/5/6)
│ 6  VCC_5V   │  ← 冗余电源
│ 7  GND      │  ← 冗余地
│ 8  NC       │  ← 预留
│ 9  RELAY_L  │  ← 继电器输出 (交流 L)
│ 10 RELAY_N  │  ← 交流 N (直通)
└──────────────┘
```

**I2C 需要上拉**（主板上已加）：
```
3.3V ──[R17 4.7k]── SDA
3.3V ──[R18 4.7k]── SCL
```

| 位号 | 元件 | 规格 | 封装 | 数量 |
|------|------|------|------|------|
| J2-J5 | 排母 | 2×5P 2.54mm 直插 | 插件 | 4 |
| R17,R18 | I2C 上拉 | 4.7kΩ | 0603 | 2 |

**强电警告**：RELAY_L 脚承载 220V AC！插槽 9、10 脚周围的 PCB 区域要和数字区域拉开爬电距离 (>6mm)，槽位之间也要隔离。

## 十-B、嘉立创 EDA 绘图建议

### 推荐图纸分页

在嘉立创 EDA 中，建议分 **6 张原理图页**：

| 页码 | 内容 | 关键元件 |
|------|------|----------|
| **第 1 页** | AC 输入 + 保护 | F1, RV1, RT1, CX1, HLK-PM01 |
| **第 2 页** | 电源树 | AMS1117-3.3, C1-C7, SS8550(PWR_HOLD) |
| **第 3 页** | ESP32-S3 最小系统 | U3, USB-C, BOOT 按键, EN 电路 |
| **第 4 页** | 继电器驱动 + 模块插槽 | ULN2003A, K1-K4, D1-D8, J2-J5 |
| **第 5 页** | HLW8032 电能计量 | U5, R9-R16, RS1, C9-C10 |
| **第 6 页** | 外设 | ST7789 接口, WS2812B, 模式按钮, 调试串口 |

### 嘉立创 EDA 操作提示

1. **搜元件**：按 `Shift+F` 打开元件搜索，直接粘贴 LCSC 编号（如 `C209903`）
2. **基础库元件**（电阻电容 LED 排针等）：直接在 EDA 左侧"元件库"面板中拖出，不需搜 LCSC
3. **封装检查**：放置元件后双击确认封装尺寸和焊盘是否与设计文档一致
4. **网络标签**：跨页连接用网络标签（Net Label），不要所有电路挤在一页
5. **ERC 检查**：画完后运行电气规则检查，确保没有单端网络

## 十一、PCB 布局指南

### 分区（从左到右）

```
┌────────┬──────────┬──────────────────────┐
│  AC    │  继电器  │       数字区          │
│ 保护   │  驱动    │                      │
│        │          │  ┌─────┐  ┌───────┐ │
│ F1 RV1 │ K1 K2    │  │ESP32│  │ 显示屏│ │
│ HLK-   │ K3 K4    │  │ S3  │  │ 接口  │ │
│ PM01   │          │  └─────┘  └───────┘ │
│        │ ULN2003A │  HLW8032  插槽1-4   │
│(开槽)  │          │  (开槽隔离 AC 侧)    │
└────────┴──────────┴──────────────────────┘
```

### 关键规则

1. **AC 区与 DC 区间 ≥ 6mm 爬电距离**，最好加开槽
2. **HLW8032 的电压采样电阻 (R9-R12)** 紧靠 HLW8032，采样走线短且远离继电器
3. **继电器线圈回路**（ULN2003A OUT → 继电器 → +5V）走线宽 > 1mm
4. **数字地和功率地** 在 HLK-PM01 输出电容处单点汇合
5. **ESP32 天线区域**（模组 PCB 天线端）下方不开铜、不铺地
6. **I2C 总线** SDA/SCL 走线平行等长，远离继电器
7. **4层板层叠**：L1(信号) / L2(GND 完整地平面) / L3(PWR 3.3V+5V) / L4(信号)
