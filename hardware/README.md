# 硬件设计文档 — 完整版 PCB (Phase 2)

## 一、系统框图

```
AC 220V 输入
  ├── 保险丝 2A (慢断)
  ├── MOV 压敏电阻 (防雷击)
  ├── NTC 热敏电阻 (防浪涌)
  └── X2 安规电容 (EMI)
         │
    [HLK-PM01] AC-DC 模块
         │ 5V/1A
    ┌────┴────┐
    │         │
[AMS1117]  [5V 继电器供电]
    │ 3.3V     │
    │         [ULN2003A ×1]
    │           │
    │         [继电器 ×4]
    │           │
    │         [模块插槽 ×4]
    │
 [ESP32-S3-WROOM-1 N16R8]
    │
    ├── SPI ── [ST7789 显示屏接口]
    ├── I2C ── [4路模块识别总线]
    ├── GPIO ── [4路 INT 中断 + 4路 RELAY]
    ├── UART ── [HLW8032 电能计量]
    ├── GPIO ── [模式按钮 + WS2812B LED]
    └── USB ─── [烧录/调试]
```

## 二、PCB 规格建议

| 参数 | 建议值 | 原因 |
|------|--------|------|
| 层数 | **4层** | AC 区域和数字区域需要隔离，4层 EMI 好做 |
| 板厚 | 1.6mm | 标准 |
| 铜厚 | 1oz (外层) / 0.5oz (内层) | 继电器走线最大 ~2A，1oz 足够 |
| 尺寸 | 约 150mm × 80mm | 参照常见排插宽度 |
| 颜色 | 绿色/黑色 | 无所谓 |
| 层叠 | SIG-GND-PWR-SIG | 标准 4 层叠构 |

### 分区要求（非常重要！）

```
┌─────────────────────────────────────────────┐
│  ⚡ AC 高压区 (左侧 40mm)                     │
│  保险丝 · MOV · NTC · X2 · HLK-PM01          │
│  ─── 安全间距 > 6mm ──────────────────────   │
│  开槽隔离槽                                   │
│  ═══════════════════════════════════════     │
│  🔌 继电器区 (中部)                           │
│  ULN2003A · 继电器×4 · 模块连接器             │
│  ═══════════════════════════════════════     │
│  🧠 数字区 (右侧)                             │
│  ESP32-S3 · HLW8032 · 显示屏 · USB           │
└─────────────────────────────────────────────┘
```

**强电区与弱电区之间必须有开槽隔离，爬电距离 > 6mm。**
